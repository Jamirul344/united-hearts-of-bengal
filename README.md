# United Hearts of Bengal — Website Bundle

## Overview
Static multi-page HTML/CSS/JS site showcasing United Hearts of Bengal. Built with accessibility, responsive design, and simple animations. Includes a volunteer form (client validation + mock POST), donate mock form, hero and counters, and sample Unsplash images.
